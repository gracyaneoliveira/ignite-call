## Getting Started

```bash
npx create-next-app@latest --use-npm
npm run dev
```
### Prisma
```bash
npm i prisma -D
npm i @prisma/client
npx prisma init --datasource-provider SQLite
npx prisma migrate dev
npx prisma studio
```
### Nookies
```bash
npm i nookies
npm i @types/cookie -D
```
### next-auth:
```bash
npm install next-auth
```
### dayjs:
```bash
npm install dayjs
```
